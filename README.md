# Responsive Personal CV Website (HTML, CSS, JavaScript)

University coursework focused on designing and building a **responsive personal CV website** from scratch using **HTML, CSS, and JavaScript**, demonstrating core web development fundamentals.

## Overview

This project is a three-page personal CV website designed to present professional information an employer would expect to see, including education, work experience, skills, interests, and a project portfolio.

A functional **Contact Me** form allows users to enquire about a potential project, specifying a date and duration, with built-in JavaScript validation.

## Pages Included

### 1) Home

Contains core CV-style information:

* Contact details (Name, Aston Email, SUN must be accurate)
* Education
* Work experience
* Skills (with programming language logos + experience rating)
* Links to professional profiles (GitHub + LinkedIn)
* Interests / hobbies

### 2) Portfolio

Shows at least **3 completed projects**, each including:

* Screenshot
* Project title
* Brief description

### 3) Contact Me

A project enquiry form that includes:

* First name
* Description
* Email + Confirm email
* Phone number
* Preferred contact method
* Project date + duration

## Design Principles Used

* **Responsive layout** (works on mobile, tablet, desktop)
* **Consistent navigation bar** on every page
* **F-pattern structure** (key info placed top-left and down the left edge)
* **Visual hierarchy** (headings, spacing, sections, emphasis)
* **Call To Action** (encourages users to visit Portfolio or Contact Me)
* Appropriate use of **icons, logos, and images**

## Skills Demonstrated

* Semantic HTML structure and accessibility basics
* Responsive CSS layout (Flexbox / optionally Bootstrap)
* Form building with validation logic
* DOM interaction using JavaScript
* Clean code practices (readability, naming, structure)
* UI structure using design principles (F-pattern, hierarchy, CTA)
